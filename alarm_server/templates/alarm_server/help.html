 Путь к фото
Место размещения фото получаемого при возникновения тревоги.
Фото делается при помощи такой команды: sudo ffmpeg -i '.$rtsp.' -y -f mjpeg -t 0.001 -s 1280x720 .'.$patch
$rtsp - путь к вашей камере по протоколу RTSP - rtsp://*IP*/user=*user*_password=* *_channel=0_stream=0.sdp?real_stream
